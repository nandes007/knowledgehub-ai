# KnowledgeHub AI — Frontend

Next.js (App Router) + TypeScript + Tailwind CSS. See the [root README](../README.md) for the project pitch and [`docs/api-contract.md`](../docs/api-contract.md) for the backend API this talks to.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
